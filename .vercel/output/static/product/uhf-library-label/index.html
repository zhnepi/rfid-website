Spec Check

What to confirm before label rollout

Frequency, label stock, printer flow, adhesive, and visible print requirements should be aligned before you lock the inlay.

Product Name UHF Library Label
Material Antenna: Aluminum etched, Substrate: Coated paper, Glue: Double-layer self-adhesive
Tag dimension 125 x 7mm, 104 x 5.5mm or customize
Antenna size 95x3mm
Supported Protocols ISO18000-6C EPC GEN2
Frequency 860-960MHz
Chip UCODE 8/9, MR6-P
Working Mode Passive
Reading Distance 6 meters
Printing graphics, numbers, barcode
Working Temperature -20 deg C to +50 deg C
Storage Temperature -20 deg C to +85 deg C
Shelf Life 1 year at 23-5 / 50% 10%RH), the vacuum bag and avoid direct sunlight exposure.
Reference Packing 10,000 pcs/ roll, roll ID: 76 mm, roll OD < 300 mm, 10 rolls /carton

Shortlist Logic

Why this label format stays in consideration

Buyers usually keep label options in play when they support the right reading mode, print flow, and packaging constraints.

Slim strip design (125x7mm) fits inside file folders without adding bulk
6-meter UHF reading distance for rapid document location and scanning
Aluminum etched antenna on coated paper substrate for reliability
Double-layer self-adhesive for secure attachment to paper and folders
High-volume roll packaging (10,000 pcs/roll) for large-scale deployment
Printable surface for graphics, numbers, and barcode encoding
